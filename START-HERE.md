# START HERE (Henro)

This folder is your DropStore ads brainstorming workspace for Claude.

## Setup (one time)
1. Make sure Claude Code is installed and you're logged in.
2. Open a terminal **in this folder** and run: `claude`
3. Claude automatically reads `CLAUDE.md` here, so it already knows the DropStore offer, the current ad numbers, the winning ads, the Andromeda rules, and where every script lives.

## Live numbers
Claude can pull **current** Meta Ads + Shopify performance any time by running `node live-perf.mjs` (read-only — it can see the numbers, never spend or change ads). Just ask it "how are we doing right now?"

## Just start talking
Examples:
- "Give me 5 new static angles we haven't tested yet."
- "Write 3 founder-ad scripts based on the First AI System winner."
- "Plan a R1,500/day campaign — exact ad sets and which creatives in each."
- "Our landing page leaks — brainstorm fixes."

## What's in here
- `CLAUDE.md` — the brief Claude reads automatically (the strategy + rules).
- `strategy/` — the current plan + real numbers (launch map, state-and-plan, master playbook).
- `creative-library/` — 43 static prompts + 14 video scripts + every angle.
- `ad-knowledge/` — ad fundamentals (ep01–10) + transcribed expert breakdowns.
- `andromeda-transcripts/` — the raw expert transcripts (deep reference).

That's it. Open Claude in this folder and brainstorm.
