# Ad Scripts — Batch #4 reload (the winning set)

> Purpose: 4 fresh creatives to load INTO the existing "First AI System" ad set
> (Batch #4 — the highest converter at ~4.87x ROAS / 4.07% CTR on the winning ad).
> Doctrine: do NOT start a new set. Reload novel creatives into the winner so it
> doesn't reset its learning, and rotate a 3rd/4th so frequency stays healthy
> (purchase probability drops sharply after ~2 impressions of the same ad).
>
> One set, up to 4 ads. All keep the PROVEN angle — *the AI builds your store for
> you* — but each leads with a DIFFERENT hook type so Meta doesn't bundle them to
> one audience (similar creatives get bundled -> frequency up, ROAS decays).
>
> Format per ad: HOOK (3-10s, the relevancy promise — ~80% never watch past it) ->
> BODY -> CTA. Shoot vertical 9:16, phone-native, no white-background slop.
> Offer to surface: DROP10 (10% off) or the active WA15 (15%). No em dashes, no hype.

---

## Why these 4 (the matrix)

The winner proves the **angle** (AI-builds-the-store) and the **format** (talking-head /
screen-record VSSL). So I hold both fixed and vary the **HOOK + persona** — the lever
the doctrine says matters most (ANGLE > OFFER > PERSONA > FORMAT, and within a proven
angle the hook is what's left to test). Four distinct hook types:

| # | Ad name | Hook type | Persona it speaks to |
|---|---------|-----------|----------------------|
| 1 | First AI System v2 (reload) | DESIRE / outcome | the winner's audience, fresh cut |
| 2 | I have no idea what I'm doing | PAIN | total beginner, scared to start |
| 3 | Stop building the store yourself | CONTRARIAN | the side-hustler who's been "researching" for months |
| 4 | Watch the AI build it in 60 seconds | PATTERN-INTERRUPT / demo | skeptic who wants proof |

Grade each hook on: a stranger gets it in 3 seconds, it's relevant, it's novel.

## Production + launch notes

- **All 4 go into the SAME ad set (Batch #4).** Do not spin up a new set — adding ads to the winner keeps its learning; a new set re-enters the learning phase.
- **Rotate, don't dump.** The doctrine: a novel 3rd/4th creative keeps purchase probability high. If the set already holds the original "First AI System" + 2 near-dead ads (POV / Student interview, ~R20-33 spend, 0 sales), consider letting those two die off and bringing these in as the fresh rotation.
- **Judge at the SET level, not per ad.** A low-ROAS ad inside a KPI-hitting set may be the top-funnel opener Meta uses in its sequence. Don't kill ads inside a winning set on last-click ROAS alone.
- **Per-ad fatigue ceiling ~R700-1.5k spend** — that's when to reload again.
- **Claims to verify before publishing:** the "15 year old, 2 days" and any "R4,000/day" / "R6,000,000+" lines come from approved comms but should get a yes from Henro/Seth before they go into *paid video* (higher scrutiny than a WhatsApp).
- **Next step after this set proves out:** clone the 2 best of these 4 as a brand-new test set for a *different* angle (e.g. founder-led VSSL, or us-vs-them vs "doing it manually"), so you're testing a new angle without touching the winner.
